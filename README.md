# Student Management System (REST API)

A robust backend REST API built with **Spring Boot** to manage student records. This project demonstrates the implementation of CRUD operations, and JPA-based database interactions.

## 🚀 Features
* **Full CRUD Operations:** Create, Read, Update, and Delete student records.
* **Email Search:** Fetch student details specifically using their unique email address.
* **Data Persistence:** Integrated with Spring Data JPA for seamless database management.

## 🛠️ Tech Stack
* **Language:** Java 21
* **Framework:** Spring Boot 4.0.5
* **Database:** MySQL (via Spring Data JPA)
* **Tooling:** Lombok, Maven, Jakarta Persistence API


## 🧪 Testing the API
You can test the endpoints using:
* Postman
* cURL
